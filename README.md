# FloodGuard

FloodGuard is an independent research project exploring the use of satellite technology and artificial intelligence to predict floods using Earth observation data.

## Project Goals
- Monitor rainfall patterns
- Detect soil moisture saturation
- Predict flood risk using AI models
- Design a conceptual satellite system for disaster monitoring

## Technologies Used
- Earth observation datasets
- Data visualization graphs
- Satellite mission concept design
